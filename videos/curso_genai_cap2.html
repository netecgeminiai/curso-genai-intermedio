1. Integrando APIs

Uso de SDKs oficiales (OpenAI / Gemini).

Hardcoding: Nunca dejes tu API Key en el código fuente. Usa `os.getenv`.

Determinismo: Usa `temperature=0` para lógica, y `1.0` para creatividad.

Roles: System, User, Assistant.

2. Pydantic y Rate Limits

Obligando a la IA a devolver un JSON perfecto.

Evitando la caída de la app por HTTP 429 (Too Many Requests).

Exponential Backoff: Esperar 1s, 2s, 4s usando `tenacity`.

3. IA como Asistente

Programación en pareja (Copilot, Cursor).

Refactorización: Convertir código heredado en limpio y tipado.

Alucinaciones: Riesgo de inyectar dependencias falsas (Hallucinated Packages).

La IA escribe, el Humano audita.

[ RIESGO MEDIO ] Vulnerabilidades lógicas.

> Solución: Implementar LLMs en CI/CD para Code Reviews automatizados.
