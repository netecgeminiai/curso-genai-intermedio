1. Agentes vs Workflows

Workflow: Determinista. Paso A -> Paso B -> Paso C.

Agente: Autónomo. Se le da un objetivo y él decide cómo cumplirlo.

Utiliza el bucle ReAct (Reasoning and Action).

Si falla, re-evalúa y prueba otra herramienta.

2. El Estándar MCP

Model Context Protocol: El nuevo estándar abierto (Anthropic/Cursor).

Permite conectar modelos de IA a fuentes de datos locales de forma segura.

Sin MCP, tendrías que programar 50 APIs para 50 apps distintas.

Arquitectura Cliente-Servidor local.

[ CLIENTE MCP ] (Ej. Cursor, Claude Desktop)
   ↕ (JSON-RPC sobre Stdio / HTTP)
[ SERVIDOR MCP ] (Tu código Python)
 ├── Tool 1: Leer Archivo Local
 ├── Tool 2: Consultar Base de Datos
 └── Tool 3: API Externa de Jira

3. Lab: Servidor MCP

Objetivo: Crear un servidor MCP en Python.

Pondremos a disposición del Agente la hora del sistema.

Usaremos decoradores `@mcp.tool()`.

Los `docstrings` son vitales: le dicen al LLM cuándo usar la herramienta.
